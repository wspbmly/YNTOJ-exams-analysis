## 问题简述
> 本题是一道典型的位运算题，非常适合初学程序设计位运算的用户入手。

给定奇数个整数，叫你在线性时间内找出其中那个唯一出现奇数次的整数。

## 问题解析
这是一个典型的位运算的题，我们只需要从头到尾“异或”一遍，最后得到的那个书就是出现了奇数次的数。这是因为“异或”有一个神奇的性质，两次异或同一个数，结果不变。
再考虑“异或”满足交换律，先异或、后异或都是一样的，因此这个方法显然是正确的。
```C++
#include <iostream>

using namespace std;

int main()
{
	int n, i, ans, m;
	
	cin >> n;
	cin >> ans;
	
	for(int i = 2; i <= n; i++)
	{
		cin >> m;
		ans ^= m;
	} 
	
	cout << ans;
	return 0;
}
```
